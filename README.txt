Aniq Load By tasid
Thank you for buying Aniq Load.

==================
Installation (web)
==================

1. Edit config.php to set User, Password and API Key.
2. Upload files.

=====================
Installation (server)
=====================

1. Go into your server.cfg file
2. Add the loading URL
   sv_loadingurl "yourdomain.com/aniqload/load.php?steamid=%s"

============
Adding Music
============

1. Go to a song on YouTube
2. Copy the ID after watch?v=
3. Paste the ID into the Admin Input

=====
Notes
=====

- The information stored on XML can be read, therefore, the API Key must be set in config.php.
- Music is only YouTube and not OGG.
